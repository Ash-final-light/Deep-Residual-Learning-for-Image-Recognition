# ResNet CIFAR-10 复现结果

## 实验配置
- 模型：ResNet-20（适配 CIFAR-10 的小型残差网络）
- 数据集：CIFAR-10
- 训练设备：cpu
- Epochs：5
- Batch Size：128
- 学习率：0.01
- 优化器：SGD(momentum=0.9, weight_decay=1e-4)
- 训练样本数：5000
- 测试样本数：1000
- 随机种子：42
- 总耗时：51.60 秒
